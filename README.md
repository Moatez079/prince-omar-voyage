# Prince Omar Voyage - Nile Cruise

Experience the magic of the Nile with Prince Omar Voyage. Luxury cruises between Luxor and Aswan.

## Features

- **Luxury Accommodations**: Comfortable and elegant rooms for your Nile journey.
- **Guided Tours**: Expert-led tours of ancient Egyptian monuments.
- **Fine Dining**: Exquisite meals featuring local and international cuisine.
- **Multilingual Support**: Available in English and Arabic.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Shadcn UI
- **Backend**: Supabase
- **State Management**: TanStack Query (React Query)
- **Routing**: React Router

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Moatez079/prince-omar-voyage.git
   cd prince-omar-voyage
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
